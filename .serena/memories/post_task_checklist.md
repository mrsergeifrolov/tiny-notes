# Post-Task Checklist

When a task is completed, ensure the following:

## 1. Code Quality
- [ ] Run linting: `npm run lint`
- [ ] Fix any ESLint errors or warnings

## 2. Build Verification
- [ ] Run build: `npm run build`
- [ ] Ensure no TypeScript compilation errors
- [ ] Check for any build warnings

## 3. Testing
- [ ] Run the dev server: `npm run dev`
- [ ] Manually test the changed functionality in browser

## 4. Code Review
- [ ] Verify code follows project conventions (see code_style.md)
- [ ] Check that optimistic updates work correctly (no UI flickering)
- [ ] Ensure Russian localization is correct for any new UI text
